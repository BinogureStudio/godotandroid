**Production ready**

Android services for Godot Engine 2.1, including:

* Godot Android
   * Share text/image
   * Network
* Google play game service
   * Authentication (linked account)
   * Leaderboards
   * Achievements
   * Snapshots
   * Player
* Facebook (Removed due to GDPR)
* Firebase
   * Authentication (anonymous)
   * User details
   * Analytics
   * Invite
   * Notification
